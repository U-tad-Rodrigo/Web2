# Skills del proceso

Las skills representan capacidades reutilizables que se aplicarán durante la práctica.

## Reglas de uso

1. Cada skill debe poder vincularse con uno o varios temas vistos.
2. Una skill no sustituye una fase; sirve como bloque operativo reutilizable.
3. Toda skill debe producir una salida verificable: esquema, tabla, contrato, decisión o checklist.
